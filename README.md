# Конфигурация сервера для автоматизации бронирования площадок на танцевальном конкурсе

## Автор
Voevodina Daria

## Программные продукты
- **Odoo 16 CRM** — управление бронированиями и базой участников
- **PostgreSQL 14** — хранение данных
- **Ansible** — автоматизация установки и настройки
- **Python** — скрипт резервного копирования
- **Nginx** — reverse proxy (порт 80 → 8069)

## Реализованные процессы
1. **Онлайн-бронирование времени** — кто, когда и какую площадку выбрал
2. **Хранение базы участников** — контактные данные коллективов в CRM
3. **Автоматическая рассылка** — уведомления о бронировании и изменениях по email

## Файлы в репозитории

 `install_odoo.yml`  Ansible playbook для полной установки сервера 
 `backup.py`  Python скрипт резервного копирования БД и файлов 
 `diagram.png`  Диаграмма развёртывания системы 

## Запуск Ansible playbook (на чистой Ubuntu)
